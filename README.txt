# Sukriti & Bibhusan — Mehndi Invitation (Self-Host Pack)

A self-contained, static invitation website. No server, no build step, no dependencies.

## What's inside
- index.html        — the invitation page
- assets/           — the artwork (toran, couple, flowers, garland)
- robots.txt        — for search engines

## How to view locally
Just double-click `index.html` (opens in any browser).

## How to host it
Upload the WHOLE folder (keep the `assets/` folder next to `index.html`) to any
static host:
- Netlify / Vercel: drag-and-drop the folder
- GitHub Pages: push the folder to a repo and enable Pages
- Any web server (Apache/Nginx/cPanel): copy the files to your public web root

That's it — open your domain and the invitation shows up.

## Editing details
Open `index.html` in any text editor. Names, date, venue and the message are
plain text near the bottom of the file (look for "Sukriti", "Bibhusan",
"10th Asar", "DreamHouse Banquets"). Change the words, save, re-upload.
